Problem/Data Domain

Yoga teachers face challenges in organizing personalized and effective class plans for diverse student needs. Managing class content, sequencing poses, and tracking plans can be time-consuming and error-prone. Existing tools like basic spreadsheets or general note-taking apps lack tailored features for yoga class planning. Dedicated apps for yoga planning, such as Tummee or Pocket Yoga, offer pre-defined sequences but often lack customization, pose management, or integration with daily teaching schedules. This lack of flexibility hinders teachers' ability to adapt plans to various yoga styles (e.g., Gentle, Flow, Restorative) or skill levels. The gap presents an opportunity for a more interactive, user-friendly tool designed specifically to address these unique needs. Our project proposes a Yoga Class Planner to empower teachers by offering seamless class creation, pose management, and search capabilities.
